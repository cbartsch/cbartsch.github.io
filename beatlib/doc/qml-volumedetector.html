Detailed Description

Detects the volume of an MP3Decoder's audio signal. It does so by using a PeakFinder on the absolute value on the signal.

It is advisible to filter the signal before using a VolumeDetector. Since this is a subclass of DirectForm2Filter, just set its DirectForm2Filter::data property to do so.

The current volume can be read from volume.

Property Documentation

filterDelayMs : int

Set this to an estimate of the DirectForm2Filter's delay time, in milliseconds. The volume will then be calculated and set that much time in advance.


startTime : int

A time interval in milliseconds since 1970/01/01 when the filtering started. This is used to adjust the detected volume for the filterDelayMs time.

You can bind this variable to MP3Decoder::startTime.


updateIntervalMs : int

Set this to a time in milliseconds the filter uses as an update interval.


[read-only] volume : real

The current detected volume, a real value between 0 and 1.
