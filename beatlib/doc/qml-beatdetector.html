Detailed Description

Property Documentation

beatHalfLifeTimeMs : real

Set this to a time in milliseconds to use as a time to fade out a detected beat's volume. This volume is used in combination with volumeDetector as a minimum volume for the next detected beat.


envelopeDetector : AudioEffect

Set this to an AudioEffect instance to apply to detect the audio signal's envelope.


minTimeDistanceMs : real

Set this to a time in milliseconds to use as a minimum time distance between detected beats.


preEffect : AudioEffect

Set this to an AudioEffect instance to apply to the audio signal before starting beat detection.


volumeDetector : VolumeDetector

Set this to an AudioEffect instance to apply to detect the audio signal's volume. This is used in combination with the beatHalfLifeTimeMs for detecting the next beat's minimum volume.
